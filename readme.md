# Responsive Hero

## Gioi thieu

Du an nay tap trung vao viec chuyen giao dien tu Figma sang `HTML` va `TailwindCSS` theo utility-first approach. Muc tieu la xay dung giao dien responsive, cau truc ro rang, de doc va de bao tri.

## Muc tieu

- Trien khai giao dien thiet ke tu Figma sang HTML + TailwindCSS
- Hieu va ap dung utility-first approach
- Viet cau truc HTML ro rang, de doc
- Su dung dung he thong layout `flex` va `grid`
- Tao responsive design tuong thich nhieu kich thuoc man hinh
- Giu code de bao tri va mo rong

## Yeu cau

### Duoc phep su dung

- HTML
- TailwindCSS utilities

### Khong duoc su dung

- Custom CSS
- Inline style
- CSS framework khac
- Component library

## Cau truc du an

- `index.html`: Trang chinh
- `image/`: Thu muc chua hinh anh

## Huong dan su dung

1. Clone hoac tai du an ve may.
2. Mo file `index.html`  bang trinh duyet.
3. Kiem tra layout tren cac kich thuoc man hinh khac nhau nhu mobile, tablet va desktop.
4. Neu can, dieu chinh HTML va cac Tailwind utilities de dam bao giao dien responsive.

## Luu y

- Giu cau truc HTML sach se, han che lap lai khong can thiet.
- Uu tien su dung `flex` va `grid` de xay dung layout.
- Tap trung vao tinh de doc va kha nang bao tri code.
